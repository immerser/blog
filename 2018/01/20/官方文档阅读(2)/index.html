名词解释

  1. Near Realtime(NRT)
    准实时搜索平台
  2. Cluster
    节点的集合
  3. Node
    每一个elastic服务代表一个节点
  4. Index
    index相当于数据库的一个database
  5. Type
    index的一个逻辑目录或者分区,对index进行分类,包含了一系列Document
  6. Document
    被索引的基本单元,相当于数据库中表的概念
  7. Shards@Replicas
    分片,用于分布式
