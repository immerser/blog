ZUUL官方文档阅读

概念

1.zuul是所有web请求的入口,作为边缘服务,能够进行 动态路由 监控 安全 弹性控制

  1. 应用场景
    1. 权限和安全
    2. 监控
    3. 动态路由
    4. 压力测试
    5. 分级负载
    6. 静态响应处理
    7. 多路弹性控制
  2. 组件
    1. zuul-core 核心模块
    2. zuul-simple-webapp 示例
    3. zuul-netflix 增加oss支持(ribbon)
    4. zuul-netflix-webapp 整合core和netflix模块

架构

通过filter实现对请求和响应的拦截和控制

  1. filter
    1. type 拦截是最经常定义的阶段
    2. execution order 执行顺序
    3. criteria 拦截需要执行的条件
    4. action 执行的动作
  2. lifecycle
    1. pre
    2. routing
    3. post
    4. error
