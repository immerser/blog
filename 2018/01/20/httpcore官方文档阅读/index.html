http core

  1. 按照内容来源可以分为三种类型的实体
    1. streamed 从流里接受
    2. self-contained 内容在内存,独立于连接,可重复生成读取
    3. wrapping 内容从其他实体获得
  2. 确保读取完流后关闭
  3. 支持的实体entity
    • BasicHttpEntity
    • ByteArrayEntity
    • StringEntity
    • InputStreamEntity
    • FileEntity
    • EntityTemplate
    • HttpEntityWrapper
    • BufferedHttpEntity
  4. 协议处理 processor
    拦截器实现,用于设置protocol,责任链方式
  5. 支持的拦截器协议(BasicHttpProcessor不是线程安全的)
    1. RequestContent
    2. ResponseContent
    3. RequestConnControl
    4. ResponseConnControl
    5. RequestDate
    6. ResponseDate
    7. RequestExpectContinue
    8. RequestTargetHost
    9. RequestUserAgent
    10. ResponseServer

6 上线文context
用于存储内部上下文变量

7 阻塞I/O模型 DefaultBHttpClientConnection
